Create a webpage containing a form that displays an input text box, a group of radio buttons, a group of checkboxes, and select items (a drop-down or datalist element); along with a script that executes form validation. Your script should validate whether the user has entered data in the input text box, has checked a radio button, has checked at least one checkbox, and has selected an option from the list of options in the select (or datalist) element. Use a submit button to invoke the validation script, so that the form is processed only when the form fields are ALL valid. If a field is invalid then display a message to the user.
